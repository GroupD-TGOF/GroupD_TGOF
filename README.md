# GroupD_TGOF

## Branching (do before working!)

* Switch to `inprogress` branch
* Branch off this branch
* Push while on your branch

```bash
git checkout -b [myname]-[mycomponent]
```

## Setup

```bash
python3 -m venv venv
pip3 install -r requirements.txt
```

## Running

* Navigate to root directory
* run `python3 main.py`
